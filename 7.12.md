# Qt 简单窗口

## Qt 对象创建

Qt 使用对象树机制管理子对象生命周期：子对象会随父对象销毁而自动释放。

构造函数：
```
QXxx(&父对象)
```
传入父对象指针，将当前对象挂载到父对象的对象树下。

随父对象释放，创建时应使用`new`运算符
子对象必须在堆上创建（`new`），父对象才能统一管理其生命周期。
栈上对象会在作用域结束时自动析构，不应作为子对象。

QWidget
    QPushButton
`QPushButton` 继承自 `QWidget`，属于按钮类控件。

## Qt 信号槽

信号槽是 Qt 对象间通信的核心机制：当信号被触发时，自动调用与之连接的槽函数。

```
connect(sender, signal, receiver, slot);
```
- `sender`：信号发送者（发出信号的对象指针）
- `signal`：信号（如 `&QPushButton::clicked`）
- `receiver`：信号接收者（处理信号的对象指针）
- `slot`：槽函数（如 `&QWidget::close`）

示例：点击按钮关闭窗口
```cpp
connect(button, &QPushButton::clicked, this, &QWidget::close);
```
